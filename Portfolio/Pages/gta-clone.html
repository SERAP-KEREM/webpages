GTA Clone

Technologies Used

Unity, C#

Description

This project is a game that emulates an open-world environment. Below are some of the key features:

  • Character Controls and Animations Creating controls and animations for player character movement. Implementing animations for running, walking, jumping, attacking, and other movements.
  • Weapon and Inventory System: Developing systems for managing weapons and other items. Adding weapons, managing inventory, switching weapons, and tracking ammo.
  • Artificial Intelligence and NPC Behavior: Developing AI systems for NPC behavior in the game. NPCs moving randomly, reacting to the player, and interacting with the environment.
  • Vehicle Controls and Mechanics: Creating systems for driving and controlling vehicles. Implementing driving mechanics, physical responses, and user controls.
  • Mission System: Providing various missions for the player character and mechanisms for completing them. Creating missions, tracking mission progress, and providing rewards.
  • Map System: Developing a map system to help players navigate the game world. Displaying locations on the map, updating explored areas.
  • Save and Load System: Creating systems for saving and loading game progress. Saving game progress and loading saved data.
  • Damage and Health System: Managing health and damage for players and NPCs. Health indicators, calculating damage, and health regeneration.
