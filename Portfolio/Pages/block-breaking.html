Block Breaking

Technologies Used

Unity, C#

Description

This project is a classic block-breaking game with the following key features:

  • Ball and Physics: Added a ball to the game, programmed to bounce off surfaces using realistic physics, enhancing the gameplay experience.
  • Paddle System: Implemented a movable paddle that players can control to direct the ball left, right, or straight, ensuring precise movement.
  • Blocks: Designed blocks with varying difficulty levels, including bonus blocks that grant extra lives, adding strategic depth to the game.
  • Levels: Created multiple levels with unique block arrangements and increasing difficulty to keep players engaged.
  • Win/Lose Conditions: Developed a system to handle win and lose conditions, such as dropping the ball or breaking all blocks in a level, providing clear objectives and feedback.
