CS Clone

Technologies Used

Unity, C#

Description

This project is a mobile FPS game inspired by Counter-Strike, featuring the following key elements:

  • Create Mobile FPS Games: Learn how to create your own PUBG or Counter-Strike inspired shooter for Android and iOS platforms.
  • Map and Environment Design: Design and optimize immersive 3D environments designed specifically for mobile devices.
  • Player Controls and Movement: Implement fluid player controls such as sprinting, jumping, and rotating with the camera.
  • Artificial Intelligence and Enemy Systems: Develop AI-driven enemies with advanced behaviors, including chasing and attacking the player.
  • Weapon Systems: Create and customize weapon systems, including shooting mechanics, reloading, and weapon attachments.
  • UI and HUD Enhancement: Design responsive UI elements such as health bars, score managers, and rifle UI for a complete gameplay experience.
  • Touch Controls Integration: Implement intuitive touch controls for player movement, aiming, and firing using joysticks and buttons.
  • Multiplayer TDM Mode: Add team deathmatch functionality with player and enemy AI teammates for competitive play.
  • Visual and Audio Effects: Enhance gameplay with muzzle flashes, sound effects, and footsteps for both players and enemies.
  • Testing and Optimization: Learn how to test and optimize your game for performance on mobile devices.
