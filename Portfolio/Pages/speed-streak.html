Speed Streak

Technologies Used

Unity, C#, AdMob

Description

This game features an endless runner where players control a character navigating through various obstacles while collecting coins. Key features include:

  • Character and Movement System: Added a character that players can control. Developed systems for running, jumping, and sliding.
  • Path Spawner System: Created a system to ensure the character runs along an endless path, maintaining continuous gameplay.
  • Obstacles and Coins: Implemented various obstacles to challenge player reflexes and placed collectible coins. Added a magnet system to attract nearby coins.
  • Shop System: Developed a shop where players can purchase different characters to enhance their gaming experience.
  • Mobile Adaptation and Ad Integration: Adapted the game for mobile devices and integrated AdMob for in-game advertisements, boosting revenue generation.
