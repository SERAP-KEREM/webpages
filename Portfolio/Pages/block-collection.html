Block Collection

Technologies Used

Unity, C#, AdMob

Description

This project is a Block Collection game with the following key features:

  • Game Design and Planning: Concept and mechanics designed for a minimalist and engaging experience. Levels and challenges were planned to ensure a balanced difficulty curve.
  • Gameplay Mechanics: Implemented core gameplay features such as block collection and obstacle navigation. Developed using Unity and C# for smooth and responsive controls.
  • Power-ups and Upgrades: Added power-ups and upgrades to enhance gameplay and provide strategic depth. Designed to improve player performance and keep the game engaging.
  • Sleek Graphics and UI: Created a clean and minimalist visual style. Designed a user interface (UI) that is both functional and aesthetically pleasing for a better user experience.
  • AdMob Integration: Integrated AdMob for monetization, including in-game ads and rewarded video ads to enhance the game's revenue potential.
  • Shader for Block Falling Mechanics: Used a custom shader to implement a hole mechanic that causes blocks to fall when they touch a hole. This feature adds a dynamic element to the gameplay, enhancing player interaction and challenge.
  • Endless Levels with Increasing Difficulty: Developed a system for endless levels where the difficulty increases progressively with each level. This system ensures continuous challenge and engagement for players.
  • Mobile Optimization: Optimized the game for mobile platforms to ensure smooth performance and a responsive experience on Android devices.
